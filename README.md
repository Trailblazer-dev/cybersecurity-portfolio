# Cybersecurity Portfolio

A portfolio for an aspiring cybersecurity professional, showcasing practical skills through CTF write-ups and project documentation. This project is built with React, Vite, and Tailwind CSS, featuring a terminal-inspired dark theme.

-   **Cybersecurity-Centric Theme** - A dark, professional theme inspired by terminal interfaces.

-   **Skills Showcase** - A section to highlight specific cybersecurity tools and competencies.
-   **Responsive Design** - Optimized for mobile, tablet, and desktop devices.
